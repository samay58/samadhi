# Milestones 0 and 1 plan

## Repository and evidence framework

Done when the generated project builds cleanly for iOS 26, Swift package tests run, the app test targets are discoverable, the preview package renders, and no production dependency exists.

## Interaction prototype

Done when every specified state can be rendered deterministically, the golden flow passes in XCUITest, the running app has been driven through semantic accessibility controls, Reduce Motion and large text have been checked, screenshots and logs are preserved, and concrete review findings have been resolved without adding scope.

## Fluid music surface

Done when no square cover art remains, the fluid field fills the ready and running surfaces, text remains legible over every deterministic palette state, Reduce Motion produces a static field, the Start hierarchy still wins in a one-second glance, and Simulator Browser captures confirm the field does not read as a generic AI animation.

Across the flow, review every state for a clear centerpiece, tactile control elevation, in-place continuity, and an obvious next action. A screen fails if it feels like a flat replacement view or if more than one motion layer asks for attention.

The current refinement pass replaces decorative run-screen rings with deterministic song-progress instrumentation, removes hard rectangular text cards in favor of edge-free tonal wells and spatial typography, and replaces repeated text wordmarks with a small original brand illustration. A native interim mark is used until final transparent artwork is supplied. The tracked ChatGPT Images prompt lives in `Docs/BRAND-ILLUSTRATION-PROMPT.md`.

## Stop condition

Stop after Milestone 1. Core Motion, adaptive audio, music import, and physical-run validation remain unimplemented and must be reported plainly.

## WHERE WE LEFT OFF

Milestones 0 and 1 satisfy their done definitions. The next implementation pass starts with real cadence acquisition and must preserve the reducer boundaries, cancellation behavior, fluid-field motion budget, open typography, and per-song progress treatment established here.
